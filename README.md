# Calculadora em C#

Projeto simples feito para praticar lógica de programação.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Porcentagem

## Como funciona
O usuário escolhe uma operação e insere dois números para obter o resultado.

## Objetivo
Praticar lógica, estruturas de repetição e condicionais em C#.
